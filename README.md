## 🚧 Stable is Under Construction

DeckOps Stable is being rebuilt from the ground up. In the meantime, please use **[DeckOps Nightly](https://github.com/GalvarinoDev/DeckOps-Nightly)** it includes everything Stable had plus LCD online play, additional games (Ghosts, Advanced Warfare, Black Ops III), multi-device support, and more.

### Install DeckOps Nightly

1. Press the Steam button → **Power** → **Switch to Desktop**
2. Open a browser and navigate to the [DeckOps Nightly](https://github.com/GalvarinoDev/DeckOps-Nightly) GitHub page
3. Download the **[DeckOps Nightly file](https://github.com/GalvarinoDev/DeckOps-Nightly/releases/download/v1/DeckOps-Nightly.desktop)**
4. Right-click the file → **Properties** → **Permissions** → tick **"Is executable"** → OK
5. Double-click it

> Your Steam games are never touched. Only files created by DeckOps are removed during uninstall.
