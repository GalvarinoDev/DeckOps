DeckOps automates the installation of iw4x, CoD4x, and Plutonium on Steam Deck. Pick your games, hit install, play.

---

## 💾 Installation

1. Press the Steam button → **Power** → **Switch to Desktop**
   - Use the right trackpad as a mouse and press **Steam + X** to open the on-screen keyboard if needed

2. Navigate to this GitHub page in a browser

3. Download the **[DeckOps installer file](https://github.com/GalvarinoDev/DeckOps/releases/download/v0.0.1-alpha/DeckOps.desktop.download)**

4. Right-click the file → **Properties** → **Permissions** → tick **"Is executable"** → OK

5. Double-click it — DeckOps installs and appears in your Steam library when done

6. Return to Gaming Mode

---

## 🗑️ Uninstall

1. Download the **[DeckOps uninstaller file](https://github.com/GalvarinoDev/DeckOps/releases/download/v0.0.1-alpha/DeckOps-uninstall.desktop.download)**

2. Right-click → **Properties** → **Permissions** → tick **"Is executable"** → OK

3. Double-click it — DeckOps is removed and all original game files are restored

> Your Steam games are never touched. Only the files DeckOps created are removed.

| Call of Duty 4: Modern Warfare | CoD4x | LCD + OLED |

## Requirements

- Steam Deck running SteamOS
- Each game installed through Steam and launched at least once
- Plutonium games require a free account at [plutonium.pw](https://plutonium.pw)
